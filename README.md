TaskFlow ◈

Gestor de tareas minimalista construido con HTML, CSS y JavaScript vanilla.


¿Qué es TaskFlow?
TaskFlow es una aplicación web para organizar tu día a día sin distracciones. Diseñada con una interfaz limpia y oscura, permite gestionar tus tareas por estado, categoría y prioridad — todo desde el navegador, sin necesidad de instalación ni cuenta.

Funcionalidades

Añadir y eliminar tareas de forma dinámica sin recargar la página
Tres estados por tarea: En progreso, Pendiente y Completada
Badges de prioridad — Alta, Media y Baja con código de colores
Categorías — Desarrollo, Deporte, Gestión e Investigación
Persistencia local — las tareas se guardan en el navegador con localStorage y sobreviven al refresco
Buscador en tiempo real que filtra tareas mientras escribes
Estadísticas y progreso semanal actualizados automáticamente en la barra lateral
Diseño responsive adaptado a móvil y escritorio
