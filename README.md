## ✦ ¿Qué es TaskFlow?

**TaskFlow** es un gestor de tareas personal construido desde cero con HTML, CSS, JavaScript puro y Tailwind CSS — sin frameworks, sin librerías, sin complicaciones.

Diseñado con una interfaz oscura y minimalista, te permite organizar tu trabajo, tus rutinas y tus proyectos en un solo lugar. Todo se guarda automáticamente en tu navegador, así que tus tareas nunca desaparecen.

---

## ⚡ Funcionalidades

- 🗂️ &nbsp;**Tres estados** — En progreso, Pendiente y Completada
- ➕ &nbsp;**Añadir tareas** con categoría y nivel de prioridad
- 🗑️ &nbsp;**Eliminar tareas** con un solo click
- 🔴 🟡 🟢 &nbsp;**Prioridades** con código de colores — Alta, Media y Baja
- 💾 &nbsp;**Persistencia total** gracias a `localStorage` — sobrevive al refresco
- 🔍 &nbsp;**Búsqueda en tiempo real** que filtra mientras escribes
- 📊 &nbsp;**Estadísticas en vivo** — progreso semanal y contadores automáticos
- 🌙 &nbsp;**Modo oscuro / claro** con toggle y preferencia guardada
- 📱 &nbsp;**Responsive** — funciona en móvil y escritorio

---

## 🛠️ Tecnologías

| | Tecnología | Uso |
|---|---|---|
| 🧱 | HTML5 | Estructura semántica |
| 🎨 | CSS3 | Animaciones, estados complejos y keyframes |
| ⚙️ | JavaScript ES6+ | DOM, eventos y LocalStorage |
| 💨 | Tailwind CSS | Sistema de diseño, utilidades y modo oscuro |

> **Tailwind CSS** se integra mediante CDN con configuración personalizada en `tailwind.config.js`, extendiendo la paleta de colores y tipografías propias del proyecto. El modo oscuro se gestiona con la estrategia `class`, activando o desactivando la clase `dark` en el elemento raíz desde JavaScript.

```
taskflow/
```

## 📚 ¿Qué aprendí con este proyecto?

- Manipulación dinámica del **DOM** con `createElement`, `appendChild` y `remove`
- Gestión de **eventos** con `addEventListener`
- **Persistencia de datos** con `localStorage`, `JSON.stringify` y `JSON.parse`
- Integración de **Tailwind CSS** y uso del prefijo `dark:` para temas dinámicos
- Diseño de sistemas de color con **variables CSS** y extensión de la config de Tailwind
- Arquitectura de una **SPA** sin frameworks
